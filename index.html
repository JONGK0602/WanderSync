Firebase 서버 직접 구축 안내

소스 코드 하단의 `defaultFirebaseConfig` 부분을 본인이 직접 만든 Firebase 프로젝트의 환경설정(Config) 값으로 교체해야 동기화 서버가 정상적으로 활성화됩니다.

외부 배포 시 서비스 이용을 위한 필수 안내

Vercel이나 GitHub 등에 본 파일을 배포하신 후에는 반드시 아래에 직접 발급받은 Gemini API Key를 입력해야 AI가 작동합니다.
